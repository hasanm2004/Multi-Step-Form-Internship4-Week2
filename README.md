# Week2 Task 02 – Multi-Step Registration Form with Validation & Local Storage

A responsive Multi-Step Registration Form developed using HTML, CSS, and JavaScript as part of the Qwetrum Technologies Internship Program 2026.

This project simulates backend-style form validation by guiding users through a three-step registration process. It includes input validation, password strength checking, data confirmation, localStorage integration, and a dynamic table to manage saved user records.

# Features:
• 3-Step Registration Process

• Personal Information Validation

• Age and Phone Number Format Validation

• Email Validation

• Password Strength Indicator

• Confirm Password Verification

• Data Confirmation Summary Before Submission

• Local Storage Data Persistence

• Dynamic Saved Entries Table

• Delete Individual Records

• Responsive User Interface


# Technologies Used:
• HTML5

• CSS3

• JavaScript (ES6)

• Local Storage API


Learning Outcomes:
• Form Validation Techniques

• Multi-Step Form Navigation

• Local Storage Management

• Dynamic DOM Manipulation

• User Input Handling

• JavaScript Event Management
